wrk 压测工具

发表于 2018-12-17 | 分类于 Devops | |

wrk -c10 -t10 -d10s -s post.lua http://localhost:9005/statistic/ ktv_reports –latency

  • -c 保持10个连接，并发数
  • -t 开多少个线程
  • -d 压测时间
  • -s Lua脚本
  • --latency 显示延迟分布
