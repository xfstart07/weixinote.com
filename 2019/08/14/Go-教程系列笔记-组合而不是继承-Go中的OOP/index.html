[Go 教程系列笔记]组合而不是继承-Go中的OOP

发表于 2019-08-14 | 分类于 Go | |

Go不支持继承，但它确实支持组合。组合的通用定义是“放在一起”。组合的一个例子是汽车。汽车由车轮，发动机和各种其他部件组成。

通过嵌入结构组成

Go中的组合可以通过将一种结构类型嵌入到另一种结构类型中来实现。
